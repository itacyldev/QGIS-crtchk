This screen allows you to select which tables you want to add to your project. By default, all the tables downloaded of the SQLite they will be added to the current project, but if you only want to add a few (for example to exclude non-geographic tables), add them to the list on the right.

The icons that annotate the names of the tables mean the following:

  • : Indicates that the table stores a geographic layer./li>
  • : Indicates that the table has a date column to store the modification date of the record.
  • : Indicates that the table has a trigger to detect changes. There is an after update that updates the record date column.
